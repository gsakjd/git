# 星尘矿野

一个原创 2D 像素风浏览器沙盒小游戏。玩家可以在横版地图里移动、跳跃、挖掘方块、收集材料，并把材料重新放回世界中。

## 仓库地址

https://github.com/gsakjd/git

## 使用方式

克隆仓库后打开 `index.html` 即可游玩：

```bash
git clone https://github.com/gsakjd/git.git
cd git
```

也可以用任意本地静态服务器打开项目目录。

## 操作

- `A` / `D` 或方向键：左右移动
- `W` / 空格 / 上方向键：跳跃
- 鼠标左键：挖掘方块
- 鼠标右键：放置当前选中的方块
- 点击底部物品栏：切换放置材料

## 当前状态

- 自动生成横版地图
- 原创像素角色和方块
- 支持挖掘、收集、放置
- 支持基础物品栏和生命值界面
